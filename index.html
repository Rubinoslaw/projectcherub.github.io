PROJECT CHERUB

PROJECT CHERUB is a codename of a distribution of TempleOS originally created by Terrence "Terry" Andrew Davis.
The ultimate goal is to create minimalist, fully functional, modern, extreme secure, simple and speed alternative for Windows and Linux. An operating system, that everyone can learn how it works in a few days of usage.

PROJECT CHERUB continues the legacy of Terry A. Davis in a way that he would like.
Project Cherub GitHub repository and ISO download link!

Features included:

- 120 FPS

Features that will be included in the future:

- 32-bit color
- SATA, NVMe, Blu-ray and USB support
- UEFI support
- GPU drivers
- Sound card drivers and music files support
- Video files support
- More readable interface
- More artistic tools

Features that wont be included in the future:

- Other screen resolutions than 640x480
- So-called "ai" things
